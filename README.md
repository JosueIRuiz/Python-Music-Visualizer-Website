# 🎵 Pro Audio Visualizer Suite - Official Website

A fully responsive promotional website and live browser-based interactive demo for the **Pro Audio Visualizer Suite**. This project bridges the gap between complex audio mathematics and highly accessible web technologies.

---

## ✨ Features

### 🌐 The Web Application (Live Demo)
* **Browser-Based Audio API:** Core audio math (like FFT) was translated into JavaScript, utilizing the Web Audio API to process live microphone input directly in the browser.
* **HTML5 Canvas Rendering:** The visualizer draws dynamically to the screen in real-time, matching the aesthetic of the original desktop application.
* **Interactive Design:** Features a custom click-to-expand interactive container that scales the visualizer seamlessly to full screen.

### 🖥️ The Promotional Website
* **Custom Frontend:** A 5-page, fully responsive website built entirely from scratch without external CSS frameworks. 
* **UI/UX Design:** Features a "retro-boxy" design language tailored to match the terminal-adjacent, classic aesthetic of a coding tool, complete with sticky navigation and integrated video media.
* **Pages Included:** Home, About the Tool, Features & Gallery, Meet the Team, and the Live Web App.

---

## 🚀 How to Run

Because this is a static website built with pure HTML, CSS, and Vanilla JavaScript, no installation, build steps, or local servers are required!

1. Clone or download this repository to your local machine.
2. Open the project folder.
3. Double-click on `index.html` to open it in any modern web browser (Chrome, Firefox, Safari, Edge).
4. Navigate to the **Live Web App** tab and click the container to grant microphone permissions and test the live visualizer.

---

## 🛠️ Tech Stack

* **Structure:** HTML5
* **Styling:** CSS3 (Custom Grid layouts, CSS Variables, Responsive UI)
* **Interactivity & Logic:** Vanilla JavaScript
* **Audio Processing:** Web Audio API
* **Graphics:** HTML5 `<canvas>`
